Obs: Aqui manda el idmenu (y el campo permiso delimita si el usuario tiene o no activado el permiso), siempre serán estos registros para cualqueir usuario, es decir rellenar con permiso 0 en caso que no tenga activado.

/*Modelo SP propuesto, pero podria mejorar el enfoque*/
CREATE PROCEDURE PCD_USUARIO (
    USUARIO VARCHAR (10),
    IDPERFIL INTEGER,
    NOMBRE VARCHAR (25),
    APELLIDO VARCHAR (25),
    DOCUMENTO VARCHAR (20),
    CONTROL INTEGER,
    RPT_USER VARCHAR (10))
RETURNS (
    MENSAJE VARCHAR (50))
AS/**/
DECLARE VARIABLE IDUSER_PERFIL VARCHAR(10);
DECLARE VARIABLE TIPO_PERFIL INTEGER;
DECLARE VARIABLE ESTADO_PERFIL INTEGER;
DECLARE VARIABLE SENTENCIA VARCHAR(350);
DECLARE VARIABLE IP VARCHAR(15);
DECLARE VARIABLE SERVER VARCHAR(100);
DECLARE VARIABLE MASTER VARCHAR(100);
DECLARE VARIABLE USER_BD VARCHAR(10);
DECLARE VARIABLE CLAVE VARCHAR(20);
DECLARE VARIABLE EXISTE_PERMISO INTEGER;
BEGIN
/*Obtenemos datos de la BD Server*/
  SELECT
    FIRST 1
    IP,
    SERVER,
    MASTER,
    USER_BD,
    CLAVE
  FROM
    CONFIGURACION_USUARIO
  INTO
    :IP,
    :SERVER,
    :MASTER,
    :USER_BD,
    :CLAVE;

/*Validaciones del Perfil de usuario*/
  select
    first 1
    iduser,
    tipo,
    estado
  from
    TIPO_USUARIO
  where
    idtipo_usuario = :idperfil
  into
    :iduser_perfil, :tipo_perfil, :estado_perfil;

  if (:iduser_perfil is null) then
  begin
    Mensaje = '1> Usuario del perfil no esta configurado';
    suspend; exit;
  end
  else
  begin
    if (:tipo_perfil not in(0,1,2)) then
    begin
      Mensaje = '2> El tipo de Perfil de usuario no esta configurado';
      suspend; exit;
    end

    if (:estado_perfil <> 1) then
    begin
      Mensaje = '3> El estado del perfil se encuentra inactivo';
      suspend; exit;
    end

    if (not exists(select * from usuario where iduser = :iduser_perfil)) then
    begin
      Mensaje = '4> Usuario no existe para el perfil elegido';
      suspend; exit;
    end
    else
    begin
      if ((select estado from usuario where iduser = :iduser_perfil) <> 1) then
      begin
        Mensaje = '5> El Usuario del perfil elegido no esta activo';
        suspend; exit;
      end
    end
  end

  if (:tipo_perfil = 0) then
  begin
    if (not exists(select * from usuarioempresa where iduser = :iduser_perfil and permisos like '%S%')) then
    begin
      Mensaje = '6> Falta configuracion de permisos en Modulo Gestion Empresarial';
      suspend; exit;
    end

    if (not exists(select * from menu_general where iduser = :iduser_perfil and permiso = 1)) then
    begin
      Mensaje = '7> Falta configuracion de menu_general en Modulo Gestion Empresarial';
      suspend; exit;
    end
  end
  else
  if (:tipo_perfil = 1) then
  begin
    if (not exists(select * from usuarioempresa where menu_gg_2 like '%1%')) then
    begin
      Mensaje = '8> Falta configuracion Modulo POS';
      suspend; exit;
    end
  end
  else
  if (:tipo_perfil = 2) then
  begin
    SENTENCIA = 'select count(*) from usuarioempresa where iduser = ? and permisos like "S"';

    FOR
    EXECUTE STATEMENT
     (SENTENCIA)
     (IDUSER_PERFIL)
    ON EXTERNAL
     :IP||':'||:MASTER
    AS
     USER     :USER_BD
     PASSWORD :CLAVE
    WITH AUTONOMOUS TRANSACTION
    INTO
      :EXISTE_PERMISO
    DO
    BEGIN
      IF (:EXISTE_PERMISO = 0) THEN
      BEGIN
        Mensaje = '9> No existe permisos para el Modulo de Contabilidad / Talento Humano';
        suspend; exit;
      END
    END
  end

  if (exists(select * from usuario where iduser = :usuario)) then
  begin
    Mensaje = '10> Ya existe usuario, imposible duplicarlo';
    suspend; exit;
  end

  if (exists(select * from usuarioempresa where iduser = :usuario)) then
  begin
    Mensaje = '11> Ya existe usuario asignado a una empresa';
    suspend; exit;
  end

  if (exists(select * from menu_general where iduser = :usuario)) then
  begin
    Mensaje = '12> Ya existe usuario asignado en Menu General';
    suspend; exit;
  end

  if (exists(select * from usuario where iduser = :usuario and documento = :documento)) then
  begin
    Mensaje = '13> Numero de documento ya existe, proceda a buscarlo por documento primeramente, considere estado inactivo o bloqueado';
    suspend; exit;
  end

/* Agregar usuario */
  in autonomous transaction do
  insert into
    usuario(
    iduser,
    nombre,
    apellido,
    pass,
    estado,
    idempresa,
    idtipo_usuario,
    documento,
    control)
  values(
    :usuario,
    :nombre,
    :apellido,
    :documento,
    1,
    1,
    :idperfil,
    :documento,
    1);

   SENTENCIA = 'INSERT INTO HISTORIAL_USUARIO(ID,USUARIO,IDOPERACION,FECHA,AUTORIZACION,OBSERVACION)' ||
   ' VALUES(GEN_ID(GEN_HISTORIAL_USUARIO,1),?, 1, CAST("NOW" AS DATE),?,"Alta de Usuario")';

  EXECUTE STATEMENT
    (SENTENCIA)
    (USUARIO,RPT_USER)
  ON EXTERNAL
    :IP||':'||:SERVER
  AS
    USER     :USER_BD
    PASSWORD :CLAVE
  WITH AUTONOMOUS TRANSACTION;

/* Agregar usuarioempresa */
  in autonomous transaction do
  insert into
    usuarioempresa(
    iduser,
    idempresa,
    permisos,
    movimientos,
    modo_print,
    talonario,
    menu_gg_2,
    permiso_gg)
  select
    :usuario,
    1,
    use.permisos,
    use.movimientos,
    use.modo_print,
    use.talonario,
    use.menu_gg_2,
    use.permiso_gg
  from
    usuarioempresa use
  where
    use.iduser = :iduser_perfil;

   SENTENCIA = 'INSERT INTO HISTORIAL_USUARIO(ID,USUARIO,IDOPERACION,FECHA,AUTORIZACION,OBSERVACION)' ||
   ' VALUES(GEN_ID(GEN_HISTORIAL_USUARIO,1),?, 1, CAST("NOW" AS DATE),?,"Alta de Usuario-Empresa")';

  EXECUTE STATEMENT
    (SENTENCIA)
    (USUARIO,RPT_USER)
  ON EXTERNAL
    :IP||':'||:SERVER
  AS
    USER     :USER_BD
    PASSWORD :CLAVE
  WITH AUTONOMOUS TRANSACTION;

/* Menu */
  in autonomous transaction do
  insert into
    menu_general (

CREATE PROCEDURE PCD_USUARIO (
    USUARIO VARCHAR (10),
    NOMBRE VARCHAR (25),
    APELLIDO VARCHAR (25),
    DOCUMENTO VARCHAR (20),
    IDPERFIL INTEGER,
    IDSUCURSAL INTEGER,
    FOTO BLOB sub_type 0 segment size 80,
    RPT_USER VARCHAR (10))
RETURNS (
    MENSAJE VARCHAR (200))
AS
DECLARE VARIABLE SENTENCIA VARCHAR(350);
DECLARE VARIABLE IP VARCHAR(15);
DECLARE VARIABLE SYSTEM VARCHAR(100);
DECLARE VARIABLE USER_BD VARCHAR(10);
DECLARE VARIABLE CLAVE VARCHAR(20);
DECLARE VARIABLE CARGO VARCHAR(150);
BEGIN
/* Inicializamos variables ---------------------------------------------------*/
  ip      = '';
  system  = '';
  user_bd = '';
  clave   = '';
  MENSAJE = 'EXITOSO';
  cargo   = '';

/* Verificamos previamente si existe USUARIO en tablas de Gestion Empresarial */
  if (exists(select * from gg_mesero where iduser = :USUARIO)) then
  begin
    Mensaje = 'Ya existe USUARIO como Vendedor en Modulo Punto de Ventas';
    suspend; exit;
  end

  if (exists(select * from usuario_sucursal where iduser = :USUARIO)) then
  begin
    Mensaje = 'Ya existe USUARIO asignado a Sucursal';
    suspend; exit;
  end

  if (exists(select * from usuario_deposito where iduser = :USUARIO)) then
  begin
    Mensaje = 'Ya existe USUARIO asignado a Deposito de Salida';
    suspend; exit;
  end

  if (exists(select * from usuario_deposito1 where iduser = :USUARIO)) then
  begin
    Mensaje = 'Ya existe USUARIO asignado a Deposito de Entrada';
    suspend; exit;
  end

  if (exists(select * from usuario_concepto where iduser = :USUARIO)) then
  begin
    Mensaje = 'Ya existe USUARIO asignado a Concepto de Movimientos';
    suspend; exit;
  end

  if (exists(select * from rh_cargo where user_system = :USUARIO)) then
  begin
    select
      ('Codigo Cargo ['||idcargo||'], '||persona) as cargo
    from(
         select
           first 1
           car.user_system,
           car.idcargo,
          ('Personal: ['||car.idpersona||'] '||per.persona) as persona
         from
           rh_cargo car
         inner join
           rh_persona per
           on (per.idpersona = car.idpersona)
         )
    (user_system, idcargo, persona)
    into
      :cargo;

    Mensaje = 'Ya existe USUARIO asignado al '|| :cargo;
    suspend; exit;
  end

/*Obtenemos datos de la BD System*/
  SELECT
    FIRST 1
    IP,
    SYSTEM,
    USER_BD,
    CLAVE
  FROM
    CONFIGURACION_USUARIO
  INTO
    :IP,
    :SYSTEM,
    :USER_BD,
    :CLAVE;

  SENTENCIA = 'SELECT MENSAJE FROM PCD_USUARIO(?,?,?,?,?,1,?);';

  EXECUTE STATEMENT
    (SENTENCIA)
    (USUARIO,IDPERFIL,NOMBRE,APELLIDO,DOCUMENTO,RPT_USER)
  ON EXTERNAL
    :IP||':'||:SYSTEM
  AS
    USER     :USER_BD
    PASSWORD :CLAVE
  WITH AUTONOMOUS TRANSACTION
  INTO
    :MENSAJE;

  IF (:MENSAJE <> 'EXITOSO') THEN
  BEGIN
    SUSPEND;
    EXIT;
  END
  ELSE
  BEGIN

    SELECT
      MENSAJE
    FROM
      PCD_OPERACIONES(
      :USUARIO,
      1,
      :RPT_USER,
      :IDSUCURSAL,
      :IDPERFIL,
      :NOMBRE,
      :APELLIDO,
      :DOCUMENTO,
      NULL) /*FOTO*/
    INTO
      :MENSAJE;

  END

  SUSPEND;

END

              CODE_ERR   = GDSCODE;
              SUSPEND;
            END

          END

          -- Usuario Deposito
          SQL_DEPOSITO = 'delete from USUARIO_DEPOSITO where IDUSER = "' || :USUARIO || '"';

          EXECUTE STATEMENT (SQL_DEPOSITO)
          ON EXTERNAL
          :IP_DESTINO || ':' || :DBASE_DESTINO AS USER :USER_DESTINO PASSWORD :PASS_DESTINO
          WITH AUTONOMOUS TRANSACTION;

          SQL_DEPOSITO = '';
          IDDEP_SALIDA = -1;
          ORDEN = -1;

          FOR
          SELECT
            DEP.IDDEPOSITO,
            COALESCE((SELECT FIRST 1
                      CASE WHEN IDSUCURSAL = 1 THEN 2 ELSE 1 END
                FROM DEPOSITO WHERE IDDEPOSITO = DEP.IDDEPOSITO),0) AS ORDEN
          FROM
            USUARIO_DEPOSITO DEP
          WHERE
            DEP.IDUSER = :USUARIO
            AND DEP.IDDEPOSITO IN(SELECT IDDEPOSITO FROM DEPOSITO WHERE COALESCE(ESTADO,0) = 1)
          INTO
            :IDDEP_SALIDA,
            :ORDEN
          DO
          BEGIN

            SQL_DEPOSITO = 'INSERT INTO USUARIO_DEPOSITO(IDUSER, IDDEPOSITO, ORDEN) VALUES(';
            SQL_DEPOSITO = :SQL_DEPOSITO || '"' || :USUARIO || '",' || CAST(:IDDEP_SALIDA AS INTEGER) || ',' || CAST(COALESCE(:ORDEN,0)  AS VARCHAR(20))|| ')';

            ESTADO = 0;

            EXECUTE STATEMENT (SQL_DEPOSITO)
            ON EXTERNAL
            :IP_DESTINO || ':' || :DBASE_DESTINO AS USER :USER_DESTINO PASSWORD :PASS_DESTINO
            WITH AUTONOMOUS TRANSACTION;

          END

          -- Usuario Deposito1
          SQL_DEPOSITO1 = 'delete from USUARIO_DEPOSITO1 where IDUSER = "' || :USUARIO || '"';

          EXECUTE STATEMENT (SQL_DEPOSITO1)
          ON EXTERNAL
          :IP_DESTINO || ':' || :DBASE_DESTINO AS USER :USER_DESTINO PASSWORD :PASS_DESTINO
          WITH AUTONOMOUS TRANSACTION;

          SQL_DEPOSITO1 = '';
          IDDEP_ENTRADA = -1;
          ORDEN = -1;

          FOR
          SELECT
            DEP.IDDEPOSITO,
            COALESCE((SELECT FIRST 1
                  CASE WHEN IDSUCURSAL = 1 THEN 2 ELSE 1 END
                FROM DEPOSITO WHERE IDDEPOSITO = DEP.IDDEPOSITO),0) AS ORDEN
          FROM
            USUARIO_DEPOSITO1 DEP
          WHERE
            DEP.IDUSER = :USUARIO
            AND DEP.IDDEPOSITO IN(SELECT IDDEPOSITO FROM DEPOSITO WHERE COALESCE(ESTADO,0) = 1)
          INTO
            :IDDEP_ENTRADA,
            :ORDEN
          DO
          BEGIN

            SQL_DEPOSITO1 = 'INSERT INTO USUARIO_DEPOSITO1(IDUSER, IDDEPOSITO, ORDEN) VALUES(';
            SQL_DEPOSITO1 = :SQL_DEPOSITO1 || '"' || :USUARIO || '",' || CAST(:IDDEP_ENTRADA AS INTEGER) || ',' || CAST(COALESCE(:ORDEN,0)  AS VARCHAR(20))|| ')';

            ESTADO = 0;

            EXECUTE STATEMENT (SQL_DEPOSITO1)
            ON EXTERNAL
            :IP_DESTINO || ':' || :DBASE_DESTINO AS USER :USER_DESTINO PASSWORD :PASS_DESTINO
            WITH AUTONOMOUS TRANSACTION;

            WHEN ANY DO
            BEGIN
              ESTADO_ERR = SQLCODE;
              CODE_ERR   = GDSCODE;
              SUSPEND;
            END

          END

          -- Usuario Concepto
          SQL_CONCEPTO = 'delete from USUARIO_CONCEPTO where IDUSER = "' || :USUARIO || '"';

          EXECUTE STATEMENT (SQL_CONCEPTO)
          ON EXTERNAL
          :IP_DESTINO || ':' || :DBASE_DESTINO AS USER :USER_DESTINO PASSWORD :PASS_DESTINO
          WITH AUTONOMOUS TRANSACTION;

          SQL_CONCEPTO = '';

          FOR
          SELECT
            IDTIPOMOVIMIENTO,
            PERMISO,
            IDTALONARIO,
            PERMISO_VARIOS,
            IDVENDEDOR,
            IDPERSONA,
            IDPLANVENTA,
            IDCONDICION
          FROM
            USUARIO_CONCEPTO
          WHERE
            IDUSER = :USUARIO
          INTO
            :IDTIPOMOVIMIENTO,
            :PERMISO,
            :IDTALONARIO,
            :PERMISO_VARIOS,
            :IDVENDEDOR,
            :IDPERSONA,
            :IDPLANVENTA,
            :IDCONDICION
          DO
          BEGIN

            SQL_CONCEPTO = 'INSERT INTO USUARIO_CONCEPTO(IDUSER,IDTIPOMOVIMIENTO,PERMISO,IDTALONARIO,PERMISO_VARIOS,IDVENDEDOR,IDPERSONA,IDPLANVENTA,IDCONDICION) VALUES(';
            SQL_CONCEPTO = :SQL_CONCEPTO || '"' || :USUARIO || '",' || CAST(:IDTIPOMOVIMIENTO AS VARCHAR(20)) || ',' || CAST(COALESCE(:PERMISO,"NULL")  AS VARCHAR(20));
            SQL_CONCEPTO = :SQL_CONCEPTO || ',' || CAST(COALESCE(:IDTALONARIO,"NULL") AS VARCHAR(20)) || ',"' || :PERMISO_VARIOS || '"';
            SQL_CONCEPTO = :SQL_CONCEPTO || ',' || CAST(COALESCE(:IDVENDEDOR,"NULL") AS VARCHAR(20)) || ',' || CAST(COALESCE(:IDPERSONA_CONCEPTO,"NULL") AS VARCHAR(20));
            SQL_CONCEPTO = :SQL_CONCEPTO || ',' || CAST(COALESCE(:IDPLANVENTA,"NULL") AS VARCHAR(20)) || ',' || CAST(COALESCE(:IDCONDICION,"NULL") AS VARCHAR(20)) || ')';

            ESTADO = 0;

            EXECUTE STATEMENT (SQL_CONCEPTO)
            ON EXTERNAL
            :IP_DESTINO || ':' || :DBASE_DESTINO AS USER :USER_DESTINO PASSWORD :PASS_DESTINO
            WITH AUTONOMOUS TRANSACTION;

            WHEN ANY DO
            BEGIN
              ESTADO_ERR = SQLCODE;
              CODE_ERR   = GDSCODE;
              SUSPEND;
            END

          END

        END --fin CONFIGURACIONES

        SQL_MIGRACION = 'SELECT RESULTADO FROM PDC_GENERA_USUARIO("USUARIO",'||:USUARIO||')';

        FOR
        EXECUTE STATEMENT (SQL_MIGRACION)
        ON EXTERNAL
        :IP || ':' || :SYSTEM AS USER :USER_BD PASSWORD :CLAVE
        WITH AUTONOMOUS TRANSACTION
        INTO
        :MIGRACION_USUARIO
        DO
        BEGIN
          -- Actualizar USUARIO en System
          EXECUTE STATEMENT
          (MIGRACION_USUARIO)
          ON EXTERNAL
          :IP_DESTINO||':'||:SYSTEM_DESTINO
          AS
          USER     :USER_DESTINO
          PASSWORD :PASS_DESTINO
          WITH AUTONOMOUS TRANSACTION;
        END

        SQL_MIGRACION = 'SELECT RESULTADO FROM PDC_GENERA_USUARIOEMPRESA("USUARIOEMPRESA",'||:USUARIO||')';

        FOR
        EXECUTE STATEMENT (SQL_MIGRACION)
        ON EXTERNAL
        :IP || ':' || :SYSTEM AS USER :USER_BD PASSWORD :CLAVE
        WITH AUTONOMOUS TRANSACTION
        INTO
        :MIGRACION_USUARIO
        DO
        BEGIN
          -- Actualizar USUARIOEMPRESA en System
          EXECUTE STATEMENT
          (MIGRACION_USUARIO)
          ON EXTERNAL
          :IP_DESTINO||':'||:SYSTEM_DESTINO
          AS
          USER     :USER_DESTINO
          PASSWORD :PASS_DESTINO
          WITH AUTONOMOUS TRANSACTION;
        END

      WHEN ANY DO
      BEGIN
        ESTADO_ERR = SQLCODE;
        CODE_ERR   = GDSCODE;
        SUSPEND;
      END

      END -- Rpl_Destino

      IN AUTONOMOUS TRANSACTION DO
      INSERT INTO HISTORIAL_USUARIO(ID,USUARIO,IDOPERACION,FECHA,AUTORIZACION,OBSERVACION)
      VALUES(GEN_ID(GEN_HISTORIAL_USUARIO,1),:USUARIO, 10, CAST('NOW' AS DATE),:RPT_USER,'Replicacion de Datos exitoso');

    END -- Operacion 10
    ELSE
/* #11 Re-Activacion de Usuario ----------------------------------------------*/
    IF (:IDOPERACION = 11) THEN
    BEGIN
      -- Usuario
      EXECUTE STATEMENT
        ('UPDATE USUARIO SET ESTADO = 1 WHERE ESTADO = 0 AND IDUSER = ?')
        (USUARIO)
      ON EXTERNAL
        :IP||':'||:SYSTEM
      AS
        USER     :USER_BD
        PASSWORD :CLAVE
      WITH AUTONOMOUS TRANSACTION;

      in autonomous transaction do
      INSERT INTO HISTORIAL_USUARIO(ID,USUARIO,IDOPERACION,FECHA,AUTORIZACION,OBSERVACION)
      VALUES(GEN_ID(GEN_HISTORIAL_USUARIO,1),:USUARIO, 11, CAST('NOW' AS DATE),:RPT_USER,'Re-Activacion de usuario exitoso');

      MENSAJE = '11';

  END /* Operacion */

  SUSPEND;

  WHEN ANY DO
  BEGIN
    MENSAJE = SQLCODE;
    SUSPEND;
  END

END




CREATE PROCEDURE PCD_BUSQUEDA_USUARIO (
    NOMBRE VARCHAR (50),
    APELLIDO VARCHAR (50),
    DOCUMENTO VARCHAR (10),
    USUARIO VARCHAR (10),
    ESTADO_USUARIO INTEGER,
    FILTRO_PERFIL VARCHAR (5))
RETURNS (
    NOMBRE_USUARIO VARCHAR (50),
    APELLIDO_USUARIO VARCHAR (50),
    IDUSER VARCHAR (10),
    ESTADO VARCHAR (8),
    DOCUMENTO_USUARIO VARCHAR (20),
    IDPERFIL INTEGER,
    PERFIL VARCHAR (30),
    FOTO BLOB sub_type 0 segment size 80)
AS/*24-03-2024 : AUTOR VLADIMIR */
DECLARE VARIABLE SENTENCIA VARCHAR(1000);
DECLARE VARIABLE EXPRESION VARCHAR(5);
DECLARE VARIABLE IP VARCHAR(15);
DECLARE VARIABLE SYSTEM VARCHAR(100);
DECLARE VARIABLE USER_BD VARCHAR(10);
DECLARE VARIABLE CLAVE VARCHAR(20);
DECLARE VARIABLE VALOR_PERFIL VARCHAR(5);
BEGIN
/* Inicializamos variables ---------------------------------------------------*/
  ip      = '';
  system  = '';
  user_bd = '';
  clave   = '';

/*Obtenemos datos de la BD System --------------------------------------------*/
  SELECT
    FIRST 1
    IP,
    SYSTEM,
    USER_BD,
    CLAVE
  FROM
    CONFIGURACION_USUARIO
  INTO
    :IP,
    :SYSTEM,
    :USER_BD,
    :CLAVE;

  IF (:NOMBRE = CAST('%' AS VARCHAR(1))) THEN
  BEGIN
    SENTENCIA = 'SELECT UPPER(USU.NOMBRE), UPPER(USU.APELLIDO), USU.IDUSER, DECODE(USU.ESTADO,0,"-",1,"+",2,"*",NULL,"/"),'||
                ' USU.DOCUMENTO, USU.IDTIPO_USUARIO, TIP.DESCRIPCION, USU.FOTO'||
                ' FROM USUARIO USU LEFT JOIN TIPO_USUARIO TIP ON (TIP.IDTIPO_USUARIO = USU.IDTIPO_USUARIO)'||
                ' WHERE USU.ESTADO LIKE ';

                IF (:ESTADO_USUARIO = 1) THEN
                    SENTENCIA = :SENTENCIA ||' 1'||
                    ' AND NOT EXISTS(SELECT * FROM TIPO_USUARIO TIU WHERE TIU.IDUSER = USU.IDUSER)';
                ELSE
                    SENTENCIA = :SENTENCIA ||' "%"';

                IF (:FILTRO_PERFIL = '0') THEN
                BEGIN
                  VALOR_PERFIL = 'NULL';
                  SENTENCIA = :SENTENCIA ||' AND USU.IDTIPO_USUARIO NOT LIKE ?';
                END
                ELSE
                IF (:FILTRO_PERFIL = '-1') THEN
                BEGIN
                  VALOR_PERFIL = '-1';
                  SENTENCIA = :SENTENCIA ||' AND COALESCE(USU.IDTIPO_USUARIO,0) = ?';
                END
                ELSE
                IF (:FILTRO_PERFIL NOT IN('0','-1')) THEN
                BEGIN
                  VALOR_PERFIL = :FILTRO_PERFIL;
                  SENTENCIA = :SENTENCIA ||' AND USU.IDTIPO_USUARIO LIKE ?';
                END

/*Ejecuta sentencia ----------------------------------------------------------*/
    FOR
    EXECUTE STATEMENT
      (SENTENCIA)
      (VALOR_PERFIL)
    ON EXTERNAL
      :IP||':'||:SYSTEM
    AS
      USER     :USER_BD
      PASSWORD :CLAVE
    WITH AUTONOMOUS TRANSACTION
    INTO
      :NOMBRE_USUARIO,
      :APELLIDO_USUARIO,
      :IDUSER,
      :ESTADO,
      :DOCUMENTO_USUARIO,
      :IDPERFIL,
      :PERFIL,
      :FOTO
    DO
    BEGIN

/*Casteo de valores ----------------------------------------------------------*/
      NOMBRE_USUARIO    = COALESCE(TRIM(:NOMBRE_USUARIO),'');
      APELLIDO_USUARIO  = COALESCE(TRIM(:APELLIDO_USUARIO),'');
      IDUSER            = COALESCE(:IDUSER,'');
      DOCUMENTO_USUARIO = COALESCE(:DOCUMENTO_USUARIO,'');
      IDPERFIL          = COALESCE(:IDPERFIL,0);
      PERFIL            = COALESCE(:PERFIL,'Desconocido');
      SUSPEND;
    END

  END
  ELSE
  BEGIN
/*Busqueda de usuario en el SYSTEM -------------------------------------------*/

    IF (:NOMBRE <> '*' AND :APELLIDO <> '*') THEN
        EXPRESION = ' AND ';
    ELSE
        EXPRESION = ' OR ';

        SENTENCIA = 'SELECT UPPER(USU.NOMBRE), UPPER(USU.APELLIDO), USU.IDUSER, DECODE(USU.ESTADO,0,"-",1,"+",2,"*",NULL,"/"),'||
                    ' USU.DOCUMENTO, USU.IDTIPO_USUARIO, TIP.DESCRIPCION, USU.FOTO'||
                    ' FROM USUARIO USU LEFT JOIN TIPO_USUARIO TIP ON (TIP.IDTIPO_USUARIO = USU.IDTIPO_USUARIO)'||
                    ' WHERE (USU.NOMBRE CONTAINING ?'||:EXPRESION||
                    ' USU.APELLIDO CONTAINING ? OR USU.DOCUMENTO CONTAINING ? OR USU.IDUSER = ?)';

/*Estado: 1 Solo activo, 0 incluye activos e inactivos -----------------------*/
                IF (:ESTADO_USUARIO = 1) THEN
                    SENTENCIA = :SENTENCIA || ' AND (USU.ESTADO = 1)'||
                    ' AND NOT EXISTS(SELECT * FROM TIPO_USUARIO TIU WHERE TIU.IDUSER = USU.IDUSER)';

                IF (:FILTRO_PERFIL = 0) THEN
                BEGIN
                  VALOR_PERFIL = 'NULL';
                  SENTENCIA = :SENTENCIA ||' AND USU.IDTIPO_USUARIO NOT LIKE ?';
                END
                ELSE
                IF (:FILTRO_PERFIL = -1) THEN
                BEGIN
                  VALOR_PERFIL = '0';
                  SENTENCIA = :SENTENCIA ||' AND COALESCE(USU.IDTIPO_USUARIO,0) = ?';
                END
                ELSE
                IF (:FILTRO_PERFIL NOT IN(0,-1)) THEN
                BEGIN
                  VALOR_PERFIL = :FILTRO_PERFIL;
                  SENTENCIA = :SENTENCIA ||' AND USU.IDTIPO_USUARIO LIKE ?';
                END

/*Ejecuta sentencia ----------------------------------------------------------*/
    FOR
    EXECUTE STATEMENT
      (SENTENCIA)
      (NOMBRE,APELLIDO,DOCUMENTO,USUARIO,VALOR_PERFIL)
    ON EXTERNAL
      :IP||':'||:SYSTEM
    AS
      USER     :USER_BD
      PASSWORD :CLAVE
    WITH AUTONOMOUS TRANSACTION
    INTO
      :NOMBRE_USUARIO,
      :APELLIDO_USUARIO,
      :IDUSER,
      :ESTADO,
      :DOCUMENTO_USUARIO,
      :IDPERFIL,
      :PERFIL,
      :FOTO
    DO
    BEGIN

/*Casteo de valores ----------------------------------------------------------*/
      NOMBRE_USUARIO    = COALESCE(TRIM(:NOMBRE_USUARIO),'');
      APELLIDO_USUARIO  = COALESCE(TRIM(:APELLIDO_USUARIO),'');
      IDUSER            = COALESCE(:IDUSER,'');
      DOCUMENTO_USUARIO = COALESCE(:DOCUMENTO_USUARIO,'');
      IDPERFIL          = COALESCE(:IDPERFIL,0);
      PERFIL            = COALESCE(:PERFIL,'Desconocido');
     SUSPEND;
    END
  END
END
